## 代理模式
代理类代理委托类，客户端不直接调用委托类的目标方法，而是通过代理类间接的调用目标方法。
这种模式增加了客户端调用目标方法的间接性，正是这种间接性提供了在目标方法调用前后增加处
理逻辑的空间。
> 注: 后续会用到Target（目标/委托类），Proxy（代理）
